# Cliengo Types

## Installation

```bash
npm install cliengo-types
```

**or**
```bash
yarn add cliengo-types
```

## Usage

```javascript
import { Account } from 'cliengo-types';

const accountData: Account;
```

## Available types
- Account
- AccountPlan
- ChatbotConfig
- i18n
- Team
- User
- Website
